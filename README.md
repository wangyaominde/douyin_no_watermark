# 抖音去水印视频下载工具

这是一个简单的抖音视频无水印下载工具，支持 **在线网页版** 和 **Python 命令行版**。

## 🌐 在线使用（推荐）

无需安装任何软件，直接访问 GitHub Pages 在线使用：

**👉 [点击访问在线下载工具](https://wangyaominde.github.io/douyin_no_watermark/)**

### 在线版使用方法：

1. 打开抖音 APP，找到想要下载的视频
2. 点击分享按钮 → 复制链接
3. 将链接粘贴到网页输入框中
4. 点击"解析"按钮
5. 解析成功后点击"下载视频"即可保存到本地

### 在线版特点：

- ✅ 无需安装任何软件
- ✅ 支持手机和电脑浏览器
- ✅ 多个解析接口可切换
- ✅ 视频预览功能
- ✅ 一键复制下载链接

## 💻 命令行版本

如果你更喜欢使用命令行，可以使用 Python 脚本版本。

### 功能特点

- **自动去水印**：下载的视频没有抖音的水印。
- **高清画质**：默认尝试下载最高画质（1080p）。
- **自动保存**：视频自动保存到 `output` 文件夹。

## 环境要求

- Python 3.6 或更高版本

## 安装步骤

1. **创建虚拟环境**（推荐）：
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # macOS/Linux
   # 或者 Windows: venv\Scripts\activate
   ```

2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

### 方式一：直接运行脚本并输入链接

运行脚本后，根据提示输入抖音分享的文本或链接：

```bash
python douyin_downloader.py
```

### 方式二：通过命令行参数传入链接

你也可以直接将分享文本作为参数传给脚本（记得给文本加上引号）：

```bash
python douyin_downloader.py "1.74 复制打开抖音，看看【赞扬Live的作品】你我山前没相见，山后别相逢！# 宋冬野 # 郭源潮... https://v.douyin.com/fQCnoZvoRt4/ PxS:/ 05/26 Z@z.gb"
```

## 输出

下载成功的视频将保存在 `output` 文件夹中，文件名为 `douyin_{video_id}.mp4`。

---

## 🔧 工作原理

```
抖音分享链接 → 重定向获取长链接 → 解析 video_id → 调用API获取无水印链接 → 下载视频
```

1. **短链接解析**：从 `v.douyin.com` 短链接通过 HTTP 重定向获取真实页面地址
2. **提取视频ID**：从页面中解析出视频的唯一标识 `video_id`
3. **获取无水印链接**：通过视频接口 API 获取高清无水印的视频直链
4. **下载保存**：流式下载视频文件并保存到本地
